# 🎬 智能视频助手 - 完整功能说明文档

> 一个功能强大的Chrome浏览器插件，支持自动跳过片头、智能识别集数、手动/自动跳转上下集、倍速播放、播放控制等实用功能。

---

## 📅 版本历史与时间节点

### v1.0 - 基础功能（2026-04-28）

**核心功能首发**

- ✅ **固定时间跳过片头**：视频播放后自动跳转到用户设定的秒数（默认60秒）
- ✅ **自动识别集数按钮**：智能识别页面上的集数按钮（如“第1集”、“第2集”）
- ✅ **自动下一集**：视频播放结束后自动点击下一集按钮
- ✅ **基础设置面板**：支持启用/禁用插件、调整跳过片头秒数
- ✅ **配置持久化**：所有设置自动保存到浏览器存储

---

### v1.1 - 智能集数识别增强（2026-04-29）

- 🔧 **URL映射识别**：支持大ID到集数的自动映射
- 🔧 **真实链接捕获**：直接从DOM获取集数的真实URL
- 🔧 **高亮样式匹配**：通过 `.active`、`.current` 等class识别当前播放集数
- 🔧 **URL近似匹配**：通过URL数字与集数列表的接近程度判断

---

### v1.2 - 手动控制按钮（2026-04-30）

- 🆕 **上一集按钮**（橙色）：点击跳转到上一集，支持循环
- 🆕 **下一集按钮**（蓝色）：点击跳转到下一集，支持循环
- 🆕 **播放/暂停按钮**（绿色）：控制视频播放状态
- 🎨 **悬停动画**：鼠标悬停时按钮放大、颜色变亮

- 🆕 **CSS选择器**：支持手动指定集数列表的选择器

### v1.5 - 日志面板开关（2026-05-03）

- 🆕 **日志面板开关**：可随时开启/关闭调试日志

- 🔧 **优先使用原生按钮**：如果页面有「上一集/下一集」按钮，插件按钮会优先点击它们
- 🔧 **降级方案**：没有原生按钮时使用集数识别逻辑

---

### v2.1 - 倍速播放功能（2026-05-05）

- 🆕 **倍速控制按钮**：右下角新增倍速控制区域
  - **减速按钮（−）**：播放速度减少 0.25x（最小 0.5x）
  - **倍速显示按钮（1.0x）**：显示当前倍速，点击切换常用倍速
  - **加速按钮（+）**：播放速度增加 0.25x（最大 3.0x）
- 🆕 **默认倍速设置**：在弹窗中可设置默认播放速度（0.5x ~ 3.0x）
- 🔧 **自动应用**：视频开始播放时自动应用默认倍速

## 🎮 当前功能详解

### 一、跳过片头

视频开始播放后，自动跳转到用户设定的秒数位置。

| 设置项 | 默认值 | 说明 |
|--------|--------|------|
| 跳过片头秒数 | 60秒 | 视频自动跳转到该时间位置（0=关闭） |

### 二、倍速播放

支持灵活的播放速度调节，满足不同观看需求。

| 按钮 | 功能 | 范围 |
|------|------|------|
| **−** | 减速 | 每次减少 0.25x，最小 0.5x |
| **1.0x** | 倍速显示/切换 | 点击循环切换：0.5 → 0.75 → 1.0 → 1.25 → 1.5 → 1.75 → 2.0 → 2.5 → 3.0 |
| **+** | 加速 | 每次增加 0.25x，最大 3.0x |

**默认倍速设置**：在弹窗中可选择默认播放速度，视频开始后自动应用。

### 三、集数识别

智能识别页面上的集数按钮，支持：

- 自动扫描页面上的集数链接
- 真实URL捕获（不依赖数字规律）
- 当前集数识别（通过URL匹配、高亮样式、数字近似）
- 可自定义识别规则（关键词、范围、选择器）

### 四、上下集跳转

- **自动跳转**：视频播放结束后自动触发
- **手动跳转**：点击右下角按钮触发
- **原生按钮优先**：如果页面有原生上下集按钮，优先使用
- **循环逻辑**：第一集↔最后一集循环跳转

### 五、播放控制

- 手动播放/暂停视频
- 按钮状态与实际播放状态实时同步

### 六、日志面板

左下角实时显示插件运行状态：

- 支持鼠标滚轮滚动
- 支持选中和复制文字
- 错误信息红色显示
- 可通过弹窗开关

### 七、设置面板

点击浏览器右上角插件图标打开：

| 设置项 | 说明 |
|--------|------|
| 启用插件 | 全局开关 |
| 跳过片头秒数 | 自定义跳过时间 |
| 默认播放倍速 | 设置默认播放速度 |
| 显示日志面板 | 调试信息开关 |
| 集数识别规则 | 高级配置（可展开） |

## 🐛 已知问题与解决方案

| 问题 | 解决方案 |
|------|----------|
| 集数识别不准确 | 调整高级设置中的关键词和范围 |
| 倍速设置不生效 | 确保视频使用标准 `<video>` 标签 |
| 控制按钮被遮挡 | 修改 `addControlButtons` 中的位置参数 |
| 页面加载卡顿 | 关闭日志面板，或更新到最新版本 |
