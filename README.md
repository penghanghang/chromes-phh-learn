# 🎬 智能视频助手 - 自动跳过片头 & 自动下一集

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)
[![Chrome Extension](https://img.shields.io/badge/Chrome-扩展-blue.svg)](https://chrome.google.com/webstore)

一款轻量级Chrome浏览器插件，支持**固定时间跳过片头**和**智能识别集数按钮自动下一集**，适用于大多数影视网站。

## ✨ 功能特性

- ⏩ **固定时间跳过片头** - 视频播放后自动跳转到指定秒数（可自定义，如60秒）
- 🔢 **智能识别集数按钮** - 自动识别网页上的集数按钮（如“1、2、3”或“第01集”），视频播放完毕后自动点击下一集
- 🎛️ **可视化设置面板** - 支持启用/禁用插件，可随时调整跳过秒数
- 💾 **配置持久化** - 所有设置自动保存到浏览器，下次打开无需重新配置
- 🔒 **隐私友好** - 不收集任何用户数据，所有逻辑在本地运行

## 📥 安装方法

### 方法一：开发者模式安装（推荐，适合未上架商店的版本）

1. 下载本仓库代码到本地
   ```bash
   git clone https://github.com/penghanghang/auto-video-helper.git
   ```
   或者直接下载ZIP压缩包并解压

2. 打开Chrome浏览器，地址栏输入 `chrome://extensions/`

3. 开启右上角的 **“开发者模式”**

4. 点击左上角 **“加载已解压的扩展程序”**

5. 选择插件所在的文件夹（包含 `manifest.json` 的那个目录）

6. 安装成功后，浏览器右上角会出现插件图标 🎬

## 🎮 使用指南

### 基础使用

1. 安装插件后，访问任意影视网站（如B站、kkys03、爱奇艺等）

2. 打开某部剧集的播放页面

3. 视频播放时会自动：
   - **跳过片头**：自动跳转到你设定的秒数（默认60秒）
   - **自动下一集**：视频播放结束后自动点击下一集的集数按钮

### 设置面板

点击浏览器右上角的插件图标 🎬，打开设置面板：

| 设置项 | 说明 |
|--------|------|
| 启用插件 | 开关插件全部功能 |
| 跳过片头秒数 | 设置跳过的秒数，例如60表示自动跳到第60秒位置 |

> 💡 设置保存后立即生效，无需刷新页面

### 支持的网站

插件通过**通用识别逻辑**工作，理论上支持所有具有以下特征的网站：

- ✅ 使用标准 `<video>` 标签播放视频
- ✅ 有明确的集数按钮（如“第1集”、“第2集”或数字按钮“1、2、3”）
- ✅ 集数按钮有 `href` 链接或可点击的DOM元素

**已验证的网站**：
- 大多数在线影视网站

**不支持的网站**：
- 使用Flash播放器的老旧网站
- 集数按钮被加密或动态生成的网站（极少数）

### 自定义集数按钮选择器

如果插件无法识别你网站的集数按钮，可以修改 `content.js` 中的 `getAllEpisodeButtons` 函数，增加适合你网站的选择器：

```javascript
const selectors = [
  'a', 'button', 'div', 'span', 'li', 'td',  // 基础标签
  '.my-episode-class',  // 添加你网站的特定class
  '[data-episode]'      // 添加你网站的自定义属性
];
```

## ❓ 常见问题

### Q: 插件无法跳过片头怎么办？

A: 请检查：
1. 设置面板中“启用插件”是否开启
2. “跳过片头秒数”是否大于0
3. 网站是否使用标准 `<video>` 标签（大部分现代网站是的）
4. 打开控制台查看是否有报错信息

### Q: 插件不会自动下一集

A: 可能原因及解决方法：
1. **集数按钮未被识别**：打开控制台，查看“找到集数按钮”的日志，如果为空或不完整，请根据上方“自定义集数按钮选择器”方法调整
2. **视频未触发ended事件**：某些网站使用自定义播放器，请在控制台手动测试 `document.querySelector('video').dispatchEvent(new Event('ended'))` 是否生效
3. **已是最后一集**：日志会显示“已是最后一集”

### Q: 页面无限刷新/循环跳转

A: 这是因为插件在视频未结束时错误触发了下一集。请确保使用的是**最新版本**的 `content.js`（依赖 `video.ended` 事件版本）。如果问题仍然存在，请在设置面板暂时关闭插件，并提交Issue说明具体网站。

### Q: 设置面板无法弹出

A: 请检查：
1. 插件是否已正确加载（`chrome://extensions/` 中无红色错误）
2. `popup.html` 和 `popup.js` 是否存在且文件名正确
3. 尝试右键点击插件图标 → “检查弹出内容”，查看控制台报错

### Q: 如何彻底卸载插件？

A: 在 `chrome://extensions/` 页面找到插件，点击“移除”即可。
